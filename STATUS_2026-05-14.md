## What shipped today

### Tooling
- `swiftx-capture diff` — markdown capture-comparison tool with LCS
  alignment + seq-prefix stripping. Surfaced the CreateGC `mask=0xc/0x8`
  divergence across xeyes AND dtcalc that pointed at empty
  `GetProperty(RESOURCE_MANAGER)`.

### Policy
- **XError-honesty policy** locked into `CLAUDE.md` Working Conventions
  and `DECISIONS.md` (2026-05-13 entry). Forces "lying is a ledgered
  exception, not a default" — every stub now needs a SHORTCUTS entry
  with a why and an exit plan, or it has to be converted to an honest
  error / real implementation.

### Infrastructure
- `Framer.XError.encode(...)` + `XErrorCode` enum
- `ServerSession.emitError(...)` helper with `[XERROR]` logging

### Validation conversions

Every handler now validates resource arguments honestly. Six helpers
cover the whole surface: `validateWindow`, `validateWindowOrRoot`,
`validateDrawTarget`, `validateGC`, `validateAtom`, `isKnownDrawable`.

| Error | Handlers converted |
| --- | --- |
| BadRequest | unknown-opcode dispatcher |
| BadDrawable | 9 poly/image-text + CopyArea + GetGeometry + QueryBestSize + PutImage |
| BadWindow | 22 window-taking handlers across Map/Unmap/Destroy/Reparent/Configure/Property/Grab/Focus/Send/Translate paths |
| BadGC | 13 draw handlers + ChangeGC/FreeGC/SetClipRectangles/SetDashes |
| BadAtom | GetAtomName, ChangeProperty, GetProperty, DeleteProperty (property + type fields) |
| BadFont | QueryFont, CloseFont, CreateGlyphCursor |
| BadPixmap | FreePixmap |
| BadCursor | FreeCursor, RecolorCursor |
| BadImplementation | cross-window CopyArea |

### Real implementations (closing known-bug categories)

- **GC clip rectangles** → `CGContext.clip` via new `withClip` helper.
  Closes xfontsel listbox / Motif text-widget bounds overflow class.
- **GC dashes** → `CGContext.setLineDash` via new `applyDashes` helper.
  Closes quickplot dashed-line class.
- **GrabKey passive grab honoring** → `findActivatablePassiveKeyGrab`
  mirrors the button-grab match algorithm. Closes quickplot accelerator
  keys.
- **SubstructureNotify chains** across Create / Map / Unmap / Destroy /
  Reparent / Configure + their *Subwindows variants. New
  `notifySubstructure(parent:build:)` helper.
- **PropertyNotify** on ChangeProperty / DeleteProperty / GetProperty
  (delete=True). New `emitPropertyNotify` helper gated on
  PropertyChangeMask.
- **VisibilityNotify** state-transition tracking. New
  `lastVisibilityState` field on WindowEntry; visibility state derives
  from `clipList` area.
- **Parent repaint + Expose on descendant unmap.** UnmapWindow and
  UnmapSubwindows now repaint parent's bg over the unmapped child's
  formerly-covered region + emit Expose to parent if subscribed. Same
  shape as the ConfigureWindow E1.5 path.
- **Root substructure**. New `rootEventMask` field lets WM-style
  clients receive top-level lifecycle events via root.
- **ConvertSelection routing** — actually shipped 2026-05-10
  (DECISIONS.md). Today: cleaned up stale SHORTCUTS entry, added unit
  tests for the three routing branches.
